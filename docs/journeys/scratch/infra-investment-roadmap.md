## Short take

If we only fund three things, I would pick 1, 2, and 3. They create the testing, observability, and architecture seams that every other investment depends on.

Since the 2026-05-20 audit, the highest-leverage **incremental** wins landed: CI ref stubs, fuzz-backed bridge/settings parsing, a real (Unity-free) integration tier, bridge listener/drain hardening, and a documented live-verify harness (canonical bundle, 13 PlayCUA scenarios, 478-test offline gate). Rank **1** still needs true multi-instance orchestration beyond `wsm-live-verify` + subagent batches; rank **2** still needs the structured observability contract and post-save-load HTTP proof.
