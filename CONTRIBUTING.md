# Contributing to WorldSphereMod3D

Governance, branch rules, and the full contributor workflow live in
[`docs/CONTRIBUTING.md`](docs/CONTRIBUTING.md).

**Quick verification before you open a PR:**

```powershell
pwsh Tools/wsm3d.ps1 doctor
pwsh Tools/wsm-live-verify.ps1
```

Optional desktop proof: `pwsh Tools/wsm-live-verify.ps1 -Live` (bridge + PlayCUA + SSIM).

Pre-merge checklist: [`docs/MERGE_CHECKLIST.md`](docs/MERGE_CHECKLIST.md).

Journey authoring only: [`docs/journeys/CONTRIBUTING.md`](docs/journeys/CONTRIBUTING.md).
